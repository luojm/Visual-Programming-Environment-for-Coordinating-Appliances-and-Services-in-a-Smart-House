ここではプログラミング環境の実行手順について記述します。

このプロジェクトはNetBeans IDE 8.0で開発し、java 1.7.0_55-b13 の中にある JavaFX のライブラリを用いて開発を進めています。
JavaFX のプロジェクトから Android のプロジェクトへの変換は https://bitbucket.org/javafxports/android/wiki/Building%20and%20deploying%20JavaFX%20Applications のサイトのものを使っています。(リンク先に環境構築手順が記述されていますので、ここには記述しません。)
動作確認をした Android 端末は Nexus10 で Android のバージョンは 4.4.2 と 4.4.4 です。
